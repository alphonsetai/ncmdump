# ncmdump

## 简介

本项目完全参考[anonymous5l/ncmdump](https://github.com/anonymous5l/ncmdump)实现，起初是为了能在windows下快速编译和运行


## 如何使用？

- 下载可执行程序[ncmdump](https://github.com/yoki123/ncmdump/releases)，将ncm文件或文件夹拖到执行程序`ncmdump`上即可
- 也可以使用命令

    ncmdump [files/dirs]
